# Phase 1 金融数据对象 Chat 测试用例

## 测试目标

通过 Chat Agent 测试 Stock、DailyQuote、Industry 对象的查询功能。

## 测试环境

- 配置文件：`configs/financial_stock_analysis.yaml`
- 数据源：Tushare Pro API
- Token：已配置环境变量 `TUSHARE_TOKEN`

---

## 测试用例分类

### 1. 基础查询测试

#### 用例 1.1：查询所有上市股票
**Chat 输入：**
```
查询所有上市的股票，显示股票代码、名称和所属行业，限制10条
```

**预期结果：**
- 返回 10 条股票记录
- 包含 ts_code、name、industry 字段
- 所有股票的 list_status 应为 'L'

#### 用例 1.2：查询特定股票信息
**Chat 输入：**
```
查询股票代码为 000001.SZ 的基本信息
```

**预期结果：**
- 返回平安银行的基本信息
- 包含完整的字段信息

#### 用例 1.3：查询股票日线行情
**Chat 输入：**
```
查询 000001.SZ 最近10天的日线行情，包括交易日期、收盘价和涨跌幅
```

**预期结果：**
- 返回最近 10 条行情记录
- 包含 trade_date、close、pct_chg 字段

#### 用例 1.4：查询所有行业
**Chat 输入：**
```
查询所有行业分类，显示行业名称
```

**预期结果：**
- 返回多个行业记录
- 包含 industry 字段
- 行业名称不重复

### 2. 过滤和筛选测试

#### 用例 2.1：按行业筛选股票
**Chat 输入：**
```
查询银行行业的所有股票，显示股票代码和名称
```

**预期结果：**
- 返回银行行业的股票列表
- 所有股票的 industry 字段为 "银行"

#### 用例 2.2：按地区筛选股票
**Chat 输入：**
```
查询深圳地区的股票，限制5条
```

**预期结果：**
- 返回深圳地区的股票
- 所有股票的 area 字段为 "深圳"

#### 用例 2.3：查询特定日期的行情
**Chat 输入：**
```
查询 000001.SZ 在 2024年1月的日线行情
```

**预期结果：**
- 返回 2024年1月 的行情数据
- trade_date 字段在 20240101-20240131 范围内

### 3. 关系查询测试

#### 用例 3.1：股票及其历史行情
**Chat 输入：**
```
查询平安银行的基本信息和最近5天的行情数据
```

**预期结果：**
- 返回股票基本信息
- 返回最近 5 条行情记录
- 行情数据的 ts_code 与股票代码匹配

#### 用例 3.2：行业内的股票列表
**Chat 输入：**
```
查询银行行业有哪些股票，并显示每只股票的最新收盘价
```

**预期结果：**
- 返回银行行业的股票列表
- 每只股票包含最新的收盘价信息

### 4. 复杂分析场景

#### 用例 4.1：行业对比分析
**Chat 输入：**
```
对比银行、保险、证券三个行业的股票数量
```

**预期结果：**
- 返回三个行业的统计信息
- 包含每个行业的股票数量

#### 用例 4.2：股票价格走势分析
**Chat 输入：**
```
分析 000001.SZ 最近30天的价格走势，计算平均价格、最高价和最低价
```

**预期结果：**
- 返回 30 天的行情数据
- 计算并显示平均价格、最高价、最低价

#### 用例 4.3：涨跌幅分析
**Chat 输入：**
```
查询银行股最近一个交易日的涨跌幅，按涨跌幅降序排列
```

**预期结果：**
- 返回银行股的涨跌幅数据
- 按 pct_chg 降序排列

#### 用例 4.4：成交量分析
**Chat 输入：**
```
查询 000001.SZ 最近10天的成交量和成交额
```

**预期结果：**
- 返回 10 天的成交量和成交额数据
- 包含 vol 和 amount 字段

#### 用例 5.2：计算平均涨跌幅
**Chat 输入：**
```
计算银行股最近一个交易日的平均涨跌幅
```

**预期结果：**
- 返回银行股的平均涨跌幅
- 计算结果准确

#### 用例 5.3：找出涨幅最大的股票
**Chat 输入：**
