# Phase 3.2: Chat Interface + ECharts Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an embedded chat interface with multi-turn dialogue, LLM function calling, and automatic ECharts visualization.

**Architecture:** ChatService orchestrates LLM function calling to MCP tools (via internal API), backend rule engine selects chart types based on data characteristics, frontend renders chat bubbles + data tables + ECharts charts.

**Tech Stack:** FastAPI, SQLAlchemy, OpenAI/Anthropic/DeepSeek SDKs, React, TypeScript, Ant Design, ECharts, echarts-for-react

---

## Implementation Summary

This plan consists of 14 tasks organized into 7 chunks. ## Task Overview

### Chunk 1: Database Models and Migration (2 tasks)

**Task 1: ChatSession and ChatMessage Models**
- Create SQLAlchemy models for chat_sessions and chat_messages tables
- Implement relationships with cascade delete
- Write unit tests for model creation
- Files: `backend/app/models/chat_session.py`, `backend/tests/test_models_chat.py`

**Task 2: Database Migration**
- Create Alembic migration script 004_add_chat_tables.py
- Define upgrade/downgrade functions for both tables
- Run migration and verify tables created
- Files: `backend/alembic/versions/004_add_chat_tables.py`, `backend/app/models/__init__.py`

### Chunk 2: Chart Engine and Pydantic Schemas (2 tasks)

**Task 3: Chart Engine**
- Implement rule-based chart type selection (line, bar, pie, scatter)
- Add data characteristic detection (time series, categorical, percentage)
- Build ECharts config generators for each chart type
- Write unit tests for chart selection logic
- Files: `backend/app/services/chart_engine.py`, `backend/tests/test_chart_engine.py`

**Task 4: Chat Pydantic Schemas**
- Define request/response schemas for chat API
- Add ChatSessionCreate, ChatMessageCreate, SendMessageRequest, SendMessageResponse
- Write validation tests
- Files: `backend/app/schemas/chat.py`, `backend/tests/test_schemas_chat.py`

### Chunk 3: ChatService with LLM Function Calling (2 tasks)

**Task 5: ChatService Core**
- Implement ChatService class with ontology context building
- Add tool schema definitions for 7 MCP tools
- Implement tool execution logic
- Add message history loading (last 20 messages)
- Files: `backend/app/services/chat.py`, `backend/tests/test_chat_service.py`

**Task 6: LLM Integration**
- Implement LLM client creation for OpenAI/Anthropic/DeepSeek
- Add function calling loop with tool execution
- Integrate chart engine for result visualization
- Handle timeouts and errors gracefully
- Files: `backend/app/services/chat.py` (modify), `backend/tests/test_chat_service.py` (extend)

### Chunk 4: Chat API Endpoints (1 task)

**Task 7: Chat API**
- Implement POST /chat/{project_id}/sessions (create session)
- Implement GET /chat/{project_id}/sessions (list sessions)
- Implement POST /chat/{project_id}/sessions/{session_id}/message (send message)
- Implement DELETE /chat/{project_id}/sessions/{session_id} (delete session)
- Register router in API __init__.py
- Files: `backend/app/api/chat.py`, `backend/app/api/__init__.py`, `backend/tests/test_api_chat.py`

### Chunk 5: Frontend Services and Models (2 tasks)

**Task 8: Frontend TypeScript Models**
- Define ChatSession, ChatMessage, SendMessageRequest/Response interfaces
- Define ChartConfig interface for ECharts
- Files: `frontend/src/models/chat.ts`

**Task 9: Chat API Service**
- Implement createSession, listSessions, sendMessage, deleteSession functions
- Use axios API client
- Files: `frontend/src/services/chat.ts`

### Chunk 6: Frontend UI Components (3 tasks)

**Task 10: ChatMessage Component**
- Create message bubble component with user/assistant styling
- Add avatar icons and timestamps
- Style with CSS for left/right alignment
- Files: `frontend/src/components/ChatMessage.tsx`, `frontend/src/components/ChatMessage.css`

**Task 11: DataTable Component**
- Create Ant Design Table wrapper for query results
- Auto-generate columns from data
- Add pagination and scrolling
- Files: `frontend/src/components/DataTable.tsx`

**Task 12: ChartRenderer Component**
- Install echarts and echarts-for-react dependencies
- Create ECharts wrapper with validation
- Handle rendering errors gracefully (fallback to table)
- Files: `frontend/src/components/ChartRenderer.tsx`, `frontend/package.json`

### Chunk 7: ChatAgent Page and Routing (2 tasks)

**Task 13: ChatAgent Page**
- Create main chat interface with session sidebar
- Implement message input with send button
- Integrate ChatMessage, DataTable, ChartRenderer components
- Add auto-scroll to latest message
- Handle loading and error states
- Files: `frontend/src/pages/ChatAgent.tsx`, `frontend/src/pages/ChatAgent.css`

**Task 14: Add Chat Route**
- Register /projects/:projectId/chat route in App.tsx
